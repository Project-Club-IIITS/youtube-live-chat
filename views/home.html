Youtube Live Chat



Guide:

Google API Key: Get a new Google API Key from Cloud Console. To use the application, you must enable Youtube Data API v3. It is suggested that you restrict the key to allow use of only Youtube Data API v3 as well.
Google Client ID: To get a OAuth Client ID, you will have to configure consent screen in Google Cloud Console. Then generate a new OAuth Key. Select Application Type as Web Application. Also, set the Authorized Javascript Domains to the domain where you will host the application. (For some reason, Google Client API only supports domains containg words, i.e. domains like 127.0.0.1 or 10.1.1.1 won't work.)
